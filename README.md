# Personal-Net-Scan
Personal network port scanner
This tool is used to connect to specific ports on a target and does the following:
- Determine if the port is open (accepting connections), closed, or filtered (blocked by firewall)
- Report the results

# Banner Grabbing
This technique has been added to the Net Scanner
- This is a reconnasissance technique used to gather information about a computer system or network service
- it involves connecting to a service and capturing the identity information of the scanned network service
- it reports the results of the service or services used
